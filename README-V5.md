# Madrador Remote V5 — 18 modules

Cette version transforme la V4 en architecture modulaire professionnelle.

## Réellement intégré

Design premium, tableau de bord, lecteur distant, overlay réseau, transfert de documents,
presse-papiers texte/images, chat, permissions détaillées, multi-écran, enregistrement
WebM, notifications internes, diagnostic local, thèmes, mode Gaming, historique et
architecture modulaire.

## Préparé mais nécessitant une infrastructure externe

- compte Madrador : backend, OAuth et base cloud ;
- mises à jour automatiques : hébergement HTTPS et signature numérique ;
- applications Android/iOS : projet mobile et signatures stores ;
- moteur Direct3D/NVENC/AMF/Quick Sync : code natif, SDK et compilation Windows ;
- connexion mondiale : serveur public et TURN.

Ces éléments sont structurés dans les dossiers `apps`, `native` et `server`, mais ne sont
pas présentés comme terminés.

## Installation

1. Décompresser le ZIP.
2. Lancer `INSTALLER-MADRADOR-V5.bat`.
3. Lancer `LANCER-EN-TEST.bat`.
4. Pour créer le Setup Windows, lancer `INSTALLER-ET-CREER-EXE.bat`.

Toutes les connexions restent visibles et nécessitent l'accord du PC distant.
